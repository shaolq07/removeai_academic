# Common AI-Flavor Checklist

Use this checklist before rewriting. Do not apply it mechanically; diagnose the text's actual failure mode.

## Ten checks

1. **Dash overuse**: If a paragraph uses more than one dash-like interruption, replace extras with commas, periods, parentheses, or separate sentences.
2. **Dense "not A but B" framing**: Keep at most 2-3 meaningful uses in a full text. Rewrite the rest as direct claims.
3. **Forced triads**: Remove artificial three-part lists. Use one, two, four, or more items only when the content requires them.
4. **Redundant sentence-openers**: Test deletion of words like "此外", "值得注意的是", "本质上", "换言之", "更重要的是". If meaning does not change, delete.
5. **Inflated endings**: Cut endings like "彰显了", "反映着", "标志着", "提供了新的启示" unless the text has earned that level of claim.
6. **Uniform rhythm**: If five or more sentences have similar length and shape, vary them. Add a short sentence where the reader needs air.
7. **Every paragraph has a maxim**: Let most paragraphs end on concrete content. Save broad reflection for moments that matter.
8. **Excess certainty**: Replace "必然", "显然", "无疑", "一定" with calibrated language when evidence is partial: "可能", "或许", "我倾向于认为", "在现有材料中".
9. **Balanced but vague stance**: Remove fence-sitting that only sounds fair. Choose the strongest supportable position.
10. **Fake precision and invented stories**: Delete fabricated numbers, friends, cases, conversion rates, or named experiences. Use real details, sourced evidence, or no detail.

## Fast scan

| Check | Question |
| --- | --- |
| Dashes | Does each paragraph use no more than one dash-like interruption? |
| Not A but B | Does the structure appear fewer than four times? |
| Triads | Does every list length come from the content rather than habit? |
| Openers | Can the sentence begin later without losing meaning? |
| Endings | Does the final verb make a real claim or only lift the tone? |
| Rhythm | Are long and short sentences mixed deliberately? |
| Maxims | Does each paragraph need a quotable ending? |
| Certainty | Is confidence matched to evidence? |
| Stance | Can the reader state the author's position? |
| Detail | Are all examples, numbers, and scenes real or clearly hypothetical? |

## Humanizing moves

- Prefer a precise noun over an abstract slogan.
- Prefer one observed detail over a stack of adjectives.
- Let transitions come from logic, not from connector words.
- Keep some asymmetry. Human writing does not always divide cleanly into equal parts.
- Use friction when appropriate: "这里真正麻烦的是...", "我不太愿意把它简单归为...", "这个判断需要更谨慎一点."
